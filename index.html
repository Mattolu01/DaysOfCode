Today, I'm writing about "Strong", "order list, unorder list & list", and "Break" tags.

The "title" tag is found in the "head" area of the HTML used to give name/title to a webpage which appears in the browser tab. "Favicon" is used to place image or icon beside the title in the browser tab.
Lameda is the title as seen on the browser tab above, and the favicon is the image beside it.

The "Break" tag, represented with "br", is used to break between words or sentences to the next line.

The "Ordered list" is used to create a list with numbers, represented as "ol".
  1. Nursery
  2. Primary
  3. Secondary
  4. Tertiary
The "Unordered list" is used to create a list item with bullets, represented as "ol".
  • Nursery
  • Primary
  • Secondary
  • Tertiary
The "list" tag is used to define each item in the list, represented as "li".
